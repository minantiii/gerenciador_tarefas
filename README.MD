# Gerenciador de Tarefas

Aplicacao simples e funcional para gerenciamento de tarefas desenvolvida com React. Permite criar, concluir, excluir e visualizar detalhes de tarefas, com dados persistidos no localStorage.

## Funcionalidades

- **Adicionar tarefas** com titulo e descricao
- **Marcar como concluida** (risca o texto da tarefa)
- **Excluir tarefas**
- **Ver detalhes** da tarefa em pagina separada
- **Persistencia local** com localStorage

## Tecnologias

- [React](https://react.dev/) 18 - Biblioteca para interfaces de usuario
- [Vite](https://vitejs.dev/) 8 - Build tool rapida para desenvolvimento
- [Tailwind CSS](https://tailwindcss.com/) 3 - Framework de estilos utilitario
- [React Router DOM](https://reactrouter.com/) 6 - Roteamento SPA
- [Lucide React](https://lucide.dev/) - Icones leves e bonitos
- [UUID](https://github.com/uuidjs/uuid) - Geracao de IDs unicos
- [ESLint](https://eslint.org/) 9 - Linter para codigo consistente
